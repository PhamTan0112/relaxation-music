# Relaxation Music 🎵

A modern web application for listening to relaxing music, ambient sounds, and focus playlists. Built with React, TypeScript, and Vite.

## ✨ Features

- 🎧 Browse relaxing music playlists
- 🔍 Search music by title or category
- ▶️ Audio player with playback controls
- ❤️ Save favorite tracks _(Coming Soon)_
- 🌙 Responsive and modern UI
- ⚡ Fast performance with Vite

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/PhamTan0112/relaxation-music.git
```

### Navigate to the project

```bash
cd relaxation-music
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 📦 Build

```bash
npm run build
```

## 🔍 Preview Production Build

```bash
npm run preview
```
